# Contributing to the Sui Development Manifesto

We welcome contributions that improve the manifesto's clarity, add relevant examples, or propose new principles.

## How to Contribute
1. Fork the repository
2. Create a branch for your changes
3. Submit a Pull Request with a clear description of your improvements

## Guidelines
- Keep the original tone and style
- Back up new principles with concrete examples
- Ensure technical accuracy
- Focus on actionable insights

## Discussion
- Open an issue for broader discussions
- Use PR comments for specific text changes
